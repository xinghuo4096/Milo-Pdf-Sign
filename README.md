# Milo-Pdf-Sign

一套给PDF文件增加电子签名的工具包

## 介绍

Milo Pdf Sign是一套给PDF文件增加电子签名的工具包，包含签名增加工具，PDF文件结构查看工具，PDF文件拆分工具，PDF压缩Stream解压工具。

## 特点

1. 闭源独立开发，拥有著作权。
2. 基于ISO PDF标准由.net C#实现，可集成到Sass服务中去。兼容性好，增加的签名在常见PDF阅读器下均可识别。
3. 基于ISO PDF标准，可以独立定义PDF内部标记，增加内部隐含标记，而不使用通用标记如：“iText-1.0.0”。
